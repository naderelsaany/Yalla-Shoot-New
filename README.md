# يلا شوت نيو | Yalla Shoot New

موقع رياضي عربي متكامل لمتابعة المباريات والأخبار الرياضية لحظة بلحظة.

## 🔧 التقنيات

- **Framework:** Next.js 16 (App Router + Turbopack)
- **Database:** Supabase (PostgreSQL + Realtime)
- **Styling:** TailwindCSS v4 (RTL + Dark Mode)
- **Language:** TypeScript
- **Hosting:** Vercel
- **Testing:** Playwright E2E

## 🚀 التشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل السيرفر محلياً
npm run dev

# بناء المشروع
npm run build

# فحص الكود
npm run lint

# تشغيل الاختبارات
npx playwright test
```

## 📁 الهيكل

```
src/
├── app/          ← صفحات الموقع و API Routes
├── components/   ← مكونات React المشتركة
├── lib/          ← Supabase Client + Translations
└── types/        ← TypeScript Types
```
